# New York v4

New York v4 is the default style system of [shadcn/ui](https://ui.shadcn.com/), serving as the foundation and base styling for all other registries. It provides a clean, modern aesthetic that can be customized and extended to create different visual variation.

## Usage

```tsx
import { YourComponent } from "@repo/ui/registry/new-york-v4/ui/your-component";
```
