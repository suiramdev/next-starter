# Overview

The `@repo/auth` package is a production-ready authentication package built on top of better-auth. It provides pre-configured, type-safe helper functions to handle authentication in your applications, with dedicated integrations for Next.js and other frameworks. 

## Installation

```bash
pnpm install @repo/auth
```

## Development

### with Next.js

Follow the [Next.js](./helpers/next-js.mdx) guide to setup the package in your Next.js application.
