# Overview

## What is next-starter and how do I get started?

next-starter is a production-ready Turborepo template for building Next.js applications. It provides a robust foundation with carefully chosen tools and patterns, allowing you to focus on building your product rather than infrastructure.

Built on years of real-world experience developing web applications, next-starter strikes the perfect balance between development velocity and code quality. It includes everything you need to build a professional SaaS application while maintaining clean, maintainable code.
